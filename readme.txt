1.git add : 提交暂存区
2.git commit -m "": 提交本地仓
3.git push origin master: 提交远程仓(第一次要加-u  : push -u)
4.git checkout -b dev : 创建并切换分区
5.git branch : 查看分区
6.git checkout : 撤销
7.git checkout name : 切换分支
8.git merge dev : 合并dev分区
9.git branch name : 创建分支
10.git checkout -b name : 创建并切换
11.git branch -d name : 删除分支
12.git merge name : 合并某分支到当前分支
13.git merge --no-ff -m "备注" name : -no-ff 表示禁用fast forward的快捷模式
14.git stash : 隐藏当前工作现场
15.git remote -v : 查看远程仓的详细信息
16.git reflog : 查看历史记录的版本号
